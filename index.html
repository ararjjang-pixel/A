탄소 사용 자동 계산 웹앱 (Prototype)

가장 현실적인 자동 수집 기능 조합을 사용한 탄소 사용량 계산 앱의 프로토타입입니다.
● 구글 위치 기록
● 대중교통 API
● 차량 OBD-II
● 전력·가스 실사용량 연동 API
이 데이터를 기반으로 시스템이 자동 계산합니다.

1. 이동 데이터 기반 탄소 계산

버튼을 누르면 '가상의 이동거리 데이터'로 탄소 배출량을 계산합니다.

2. 전력/가스 사용 기반 자동 계산

실제로는 한전·도시가스 API와 연동되지만, 여기서는 임의 데이터로 계산합니다.

3. 오늘의 총 탄소 사용량

위의 두 항목 데이터를 합산합니다.
